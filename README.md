# UNIT

Сайт-визитка UNIT (Frontend-приложение)

Стек технологий:

- React
- TypeScript
- Vite
- TanStack Query
- axios
- Tailwind CSS

## Запуск проекта

1. Склонировать репозиторий

```bash
git clone https://github.com/TheStoryOfaGirl/Cringe.git
cd Cringe
```

2. Установить зависимости

```bash
npm install
```

3. Запустить проект

```bash
npm run dev
```
